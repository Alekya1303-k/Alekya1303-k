## 🔬 Featured Project

### 🍓 [Frozen Fruit Digital Twin Framework](https://github.com/Alekya1303-k/frozen-fruit-digital-twin)
> Agent-Based Digital Twin for Frozen Fruit Early Detection and Quality Prediction

An end-to-end AI pipeline combining:
- **EfficientNet-B0 CNN** — 97% accuracy image classifier with Grad-CAM explainability
- **BiLSTM + Bahdanau Attention** — 90-day quality forecasting from IoT sensor streams
- **Multimodal Fusion** — 1536-dim Digital Twin state · RMSE=0.0567 · R²=0.9459
- **Hybrid Agent** — 5-tier autonomous alert system with confidence scoring & trend detection
- **LLM Dashboard** — Claude Sonnet API generating plain-English quality reports every 15 min on Streamlit
- **Arrhenius Kinetics Simulation** — Physics-based frozen degradation dataset generation (novel contribution)

`Python` `TensorFlow` `BiLSTM` `EfficientNet` `Claude API` `Streamlit` `Grad-CAM`
